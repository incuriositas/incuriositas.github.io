빨간모자의 진실 RPG게임

2017

유니티라는 게임엔진을 이용해 원래 목적은 몬스터를 죽이면서 각 단계를 깨는 RPG게임이었으나 시간분배문제로 인해 몬스터를 피해 오래살아남는 게임이 되어버린 '빨간모자의 진실'게임을 제작해보았다. 게임을 만들면서 나는 점프대 생성 및 스크립트작성과, 각 언덕들과 언덕과 언덕을 지날 수 있는 외나무다리를 생성해 외나무 다리가 회전하는 스크립트를 작성하였다. 후에 유니티를 좀 더 배우고 싶어 18년도 1학기에 학술부장이 유니티강좌를 하는 것을 신청하여 듣기도 하였다.

  • 개발언어: C#
  • 맡은 역할: 장애물 제작 및 스크립트
  • Client: Game
  • Category: project

학식평가 사이트

github repository

제주대학교에서는 학식평가를 그날에 메뉴을 적어놓은 판에 스티커를 붙여 평가하는 시스템입니다. 저는 학식의 평가를 다른 위치에서 보기에 힘들어, 학식은 맛이없다라는 생각에 학식을 먹을 생각을 하지조차않았는데 가끔 맛있었다는 평가가 들려오기도 했습니다. 그래서 저는 평가서비스를 제공해주는 사이트가 있으면 좋겠다고 생각했으며, 그날의 메뉴만 평가하는 것이 아니라, 메뉴 선정 및 다른 서비스 또한 모두 포함하는 평점을 볼 수 있도록 사이트를 제작하였습니다. 학식데이터를 크롤링하여 메뉴를 보여주며, 카테고리별 별점확인기능 , 비밀번호기반의 게시글 작성 및 수정을 할 수 있는 사이트입니다.

  • 개발언어: Java
  • Client: WEB
  • Category: Project
